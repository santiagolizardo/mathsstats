# {{ page.title }}

Typically used for time intervals between succesive events.

<dl>
    <dt>Notation</dt>
    <dd>$$M(\lambda)$$</dd>

    <dt>Range</dt>
    <dd>$$x \ge 0$$</dd>

    <dt>Probability mass function p(x)</dt>
    <dd>
    $$
    \lambda e^{-\lambda x}
    $$
    </dd>

    <dt>Mean</dt>
    <dd>$$\frac{1}{\lambda}$$</dd>

    <dt>Mode</dt>
    <dd>$$0$$</dd>

    <dt>Variance</dt>
    <dd>$$\frac{1}{\lambda^2}$$</dd>
</dl>
