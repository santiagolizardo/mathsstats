# {{ page.title }}

Typically used for measuraments of z-intervals, sample size estimation.

<dl>
    <dt>Notation</dt>
    <dd>$$N(0, 1)$$</dd>

    <dt>Range</dt>
    <dd>$$-\infty < x < \infty$$</dd>

    <dt>Probability mass function p(x)</dt>
    <dd>
    $$
    \frac{1}{\sqrt{2 \pi}} exp\left(-\frac{1}{2} x^2\right)
    $$
    </dd>

    <dt>Mean</dt>
    <dd>$$0$$</dd>

    <dt>Median</dt>
    <dd>$$0$$</dd>

    <dt>Mode</dt>
    <dd>$$0$$</dd>

    <dt>Variance</dt>
    <dd>$$1$$</dd>
</dl>
