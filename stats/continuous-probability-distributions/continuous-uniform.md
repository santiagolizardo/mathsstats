# {{ page.title }}

Typically used for equally likely values on the interval [a, b]; flat priors.

<dl>
    <dt>Notation</dt>
    <dd>$$U(a, b)$$</dd>

    <dt>Range</dt>
    <dd>$$a \le x \le b$$</dd>

    <dt>Probability mass function p(x)</dt>
    <dd>
    $$
    \frac{1}{b - a}
    $$
    </dd>

    <dt>Mean</dt>
    <dd>$$\frac{a + b}{2}$$</dd>

    <dt>Variance</dt>
    <dd>$$\frac{(b - a)^2}{12}$$</dd>
</dl>
