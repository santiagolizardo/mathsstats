# {{ page.title }}

A *time series* is a collection of observations $$X_t$$ on some random variable $$X$$ at equally-spaced times $$1, 2, \ldots, t, t+1, \ldots$$. A **time plot** is a graph of the observed values $$x_t$$ against $$t$$.
